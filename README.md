## 🧬 About Me

I'm **Sanjay**, an **ECE undergraduate** with a dual focus on **Quantum Hardware Engineering** and **VLSI Chip Design** — two fields that converge in the control electronics stack for superconducting quantum computers.

My work spans from **transmon qubit layout and EM simulation** at the quantum hardware level, all the way down to **RTL design, FPGA implementation, and full ASIC flows** for the classical control side. I'm currently building expertise in **RFSoC-based control electronics** — the critical bridge between room-temperature digital logic and millikelvin qubit systems.

- ⚛️ Designing **Transmon qubits** — Hamiltonian, $E_C / E_J$ tuning, anharmonicity, frequency extraction via Q3D
- 🏗️ Active project: **4-Qubit Flip-Chip Transmon Processor** — layout, coupling & readout design
- 📡 Deep in the **readout chain**: Resonator → TWPA → HEMT → IQ mixer → state discrimination
- 🔌 Building **RFSoC IP blocks** on ZCU111 — ADC/DAC, RF Data Converter, qubit control pulse generation
- 🖥️ Full **VLSI ASIC flow**: RTL → Genus → Innovus → Tempus → Virtuoso → Synopsys Custom Compiler
- 🎯 Goal: **Quantum Hardware Engineer** working on superconducting qubit systems and their control stack
